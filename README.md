# Exam Seating Arrangement System

## Technology Stack
- **Frontend**: React.js (Web) + JavaFX (Desktop)
- **Backend**: Java Spring Boot with REST APIs
- **Database**: MySQL
- **Deployment**: AWS/Heroku ready
- **AI Features**: Intelligent seat allocation algorithm

## Project Structure
```
ExamSeatingArrangements/
├── backend/                 # Spring Boot API
├── frontend-web/           # React.js Web App
├── frontend-desktop/       # JavaFX Desktop App
├── database/              # SQL scripts
└── deployment/            # Docker & deployment configs
```

## Features
- Student management
- Exam hall configuration
- AI-based seat allocation
- Anti-cheating optimization
- REST API endpoints
- Responsive web interface
- Desktop application
